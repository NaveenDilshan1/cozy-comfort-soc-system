# Cozy Comfort SOC System

A Service-Oriented Computing (SOC) based supply chain management system for Cozy Comfort.

## 🚀 Tech Stack
- Backend: .NET Web API
- Frontend: React.js
- Architecture: Microservices / SOC

  https://github.com/NaveenDilshan1/cozy-comfort-soc-system/raw/main/home.png

## 📦 Features
- Inventory Management
- Order Processing
- Supplier Management
- API-based communication between services

## 📁 Project Structure
CozyComfort.API → Backend services  
CozyComfort.Frontend → React frontend  

## ▶️ How to Run

### Backend
cd CozyComfort.API
dotnet run

### Frontend
cd CozyComfort.Frontend
npm install
npm start
